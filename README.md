# 方寸工具箱 - 思源笔记插件

一款为 [思源笔记](https://b3log.org/siyuan/) 打造的便捷工具箱插件，提供悬浮快捷按钮和自定义配置功能。

[![GitHub](https://img.shields.io/badge/GitHub-仓库-orange)](https://github.com/ycbhk/fangcun-toolbox)

## 📖 简介

方寸工具箱是一个轻量级但功能强大的思源笔记辅助工具，通过在屏幕右侧显示悬浮按钮，让你能够快速访问常用功能，提升笔记效率。

## ✨ 主要功能

### 1. 悬浮快捷按钮
- 🎯 **屏幕右侧悬浮**：不遮挡笔记内容，随时可用
- 🎨 **可自定义颜色**：支持修改背景色和图标颜色
- 📱 **跨端适配**：PC 端和移动端自动适配
- 🖱️ **拖拽定位**：支持拖拽调整位置，8px 防误触阈值

### 2. 展开面板
- 📋 **快捷操作集合**：一键打开常用功能面板
- 🔝 **靠右上方对齐**：智能定位，不被悬浮按钮遮挡
- ⚡ **快速访问**：复制、收藏、导出等常用功能

### 3. 自定义配置
- ⚙️ **设置界面**：简洁美观的 TAB 式设置界面
- 🎛️ **灵活控制**：启用/禁用悬浮按钮
- 🎨 **颜色定制**：每个按钮都可自定义配色
- 💾 **自动保存**：配置实时保存，重启生效

## 🚀 使用方法

### 安装方法

#### 方式一：思源笔记集市安装（推荐）
1. 打开思源笔记
2. 点击顶部菜单 `设置` → `集市`
3. 搜索 "方寸工具箱"
4. 点击安装

#### 方式二：手动安装
1. 下载插件源码
2. 将插件文件夹复制到 `<工作空间>/data/plugins/` 目录
3. 重启思源笔记
4. 在 `设置` → `插件` 中启用插件

### 基本操作

#### 1. 启用插件
```
设置 → 插件 → 方寸工具箱 → 启用
```

#### 2. 打开悬浮按钮
```
设置 → 插件 → 方寸工具箱 → 基础功能 → 启用悬浮按钮
```

#### 3. 使用悬浮按钮
- **单击**：打开展开面板
- **拖拽**：调整悬浮按钮位置
- **再次单击**：关闭展开面板

#### 4. 配置按钮
1. 点击设置界面中的 ⚙️ 图标
2. 进入悬浮按钮详细设置
3. 可以：
   - 开启/关闭特定按钮
   - 修改按钮背景颜色
   - 修改按钮图标颜色
4. 所有修改自动保存
